# techne.coop

Source for [techne.coop](https://techne.coop) — the virtual home of **Techne Studio**, a DBA of **RegenHub, LCA** (Colorado Limited Cooperative Association, SOS #20261163853).

Hosted on GitHub Pages from the `main` branch. Custom domain: `techne.coop`. HTTPS enforced.

---

## What This Site Is

Techne is the studio brand of RegenHub, LCA. The co-op's public benefit: cultivating scenius — the collective intelligence that arises when talented people share a commons.

`techne.coop` is the virtual co-op community: membership information, the cooperative's public documents, tools for co-op members, and the interface between Techne Studio and the world.

---

## Intended Structure

```
/                   → Landing page (cooperative home, mission, membership CTA)
/about/             → Cooperative overview, organizers, legal entity
/membership/        → Membership classes, how to join, dues and equity
/legal/             → Formation documents (links to techne.institute/legal/)
/journal/           → Links out to journal.techne.coop
/work/              → Studio work, ventures, projects
/assets/            → Shared CSS tokens, fonts, JS
```

The design system follows the Techne visual language: Libre Baskerville (serif), IBM Plex Mono, warm dark palette (`#0F0F12` base), gold accent (`#C4956A`).

This is a cooperative project. Proposed changes go through the Workshop coordination process at the co-op.us Workshop before merge.
